# Sshot (In development)

Sshot is a simple screenshot tool for Linux. It is written in C and SDL3. It is designed to be simple and easy to use. It is not meant to be a full-featured screenshot tool, but rather a simple tool for taking screenshots.

## Features and Features in development
- Take screenshots of the entire screen or a selected area
- Save screenshots in PNG format
- Save screenshots to a specified directory
- Copy screenshots to the clipboard
- Simple and easy to use interface
